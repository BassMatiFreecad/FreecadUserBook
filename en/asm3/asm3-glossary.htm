Assembly 3 Glossary

Tags

Freecad, Assembly 3, Glossary.

Assembly (Model Tree Node)

A container for Parts, Connectors and Constraints.

Assembly Constraint

Represents a geometrical relationship between Constraint Connectors. Associated with a Constraint node in the Model Tree.

Connector (Model Tree Node)

Child node of the Connectors (alias Elements) node, alias for the Element node. Represents a Constraint Connector, and links to a geometry element.

The child node of a Constraint is a Link to a Connector in the Connectors collection.

Connector Link (Model Tree Node)

Child node of a Constraint node. Points to a Connector node.

Connectors (Model Tree Node)

Child node of the Assembly node. Alias for the Elements node.

Constraint Connector

End point for an Assembly Constraint. Binds to a geometry element of a Part in the Parts collection of the Assembly or a Subassembly. Represented by a Connector node in the Model Tree.

Constraint (Model Tree Node)

A child node of the Constraints node. Encapsulates the properties of an Assembly Constraint defined between two Connectors.

The child nodes of a Constraint node are Links to the ConnectorLink nodes.

Constraints (Model Tree Node)

A child node of the Assembly node. Parent node of the Constraint nodes.

Create Assembly

Toolbar: Assembly 3
Menu: Assembly3 > Create Assembly
Shortcut: A N

Creates a new Assembly object in the active document.

Element (Model Tree Node)

Alias for Connector. Represents a Constraint Connector in the Model Tree.

Elements (Model Tree Node)

Group node for Connector alias Element nodes.

Part (Model Tree Node)

Represents a Freecad object that generates a geometry, such a Part Design Body, Freecad structural Part, an object generated by the Part Workbench, or an Assembly object. (The list is non-exhaustive.) Constraint Connectors bind to a geometry element generated by a Part or a Part in a subassembly.

Parts (Model Tree Node)

Child node of the Assembly node, Parent for Part nodes.

Subassembly

An Assembly can contain another Assembly in its Parts collection. In other words, assemblies can be nested. So a subassembly is an Assembly in a parent Assembly.
