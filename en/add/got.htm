Glossary of Terms

Tags

Freecad, Glossary.

Child Node

A node in the Model Tree that has a parent node.

Collection

(1) The child nodes of a group node in the Model Tree.

(2) A group node in the Model Tree.

Component

Synonyms: Part.

(1) A (re-usable) unit of construction.

Folder

Alias: Group.

Group

Structural element, a node in the Model Tree, used as a container for related components.

Alias: Folder.

Group Node

Synonyms: Collection.

A Node in the Model Tree that can have child nodes.

Library

Alias: Repository.

Model Tree

A hierarchical representation of the design in the form of a collapsible tree.

Node

An structural element in the Model Tree.

Parent Node

A node in the Model Tree that has a child node.

Part

Synonyms: Component.

(1) A (re-usable) unit of construction.

(2) A child node of the Parts collection in an Assembly node.

Repository

A container for (re-usable) parts or components.

Alias: Library.

An internal repository is just a folder in a project file.
