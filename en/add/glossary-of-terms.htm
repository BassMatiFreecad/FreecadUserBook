Folder

Alias: Group.

Group

Structural element, a node in the Model Tree, used as a container for related components.

Alias: Folder.

Library

Alias: Repository.

Model Tree

A hierarchical representation of the design in the form of a collapsible tree.

Repository

A container for (re-usable) parts or components.

Alias: Library.

An internal repository is just a folder in a project file.
