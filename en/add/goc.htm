Mark to Recompute

Sets a flag on an unmodified component to be recomputed with the Refresh command.

Where to find: Model Tree node, context menu.

Related: Refresh.

Related: Recompute Object.

Refresh

Recalculates the current document.

Where to find: Menu Edit > Refresh, keyboard F5.

Similar: Recompute Object.

Related: Mark to Recompute.

Recompute Object

Recalculates the selected object.

Similar: Refresh.
